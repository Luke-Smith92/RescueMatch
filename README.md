# RescueMatch

## Project Overview
RescueMatch is a pet rehoming platform that connects approved rescue centres with members of the public looking to adopt animals.

Users can browse available pets and match with animals they are interested in. Once matched, contact details for the rescue centre are revealed.

---

## Features (Planned)
- View available pets
- Match with pets
- Reveal rescue contact details
- Add, edit, and delete pet listings (admin)

---

## Technologies Used
- Python (Flask)
- HTML
- CSS
